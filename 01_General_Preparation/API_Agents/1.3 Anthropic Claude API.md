# 🤖 Configuration & Connection Guide: Anthropic Claude API

This guide covers account creation, API key generation, credential validation, and basic integration for the **Anthropic Claude API** — following the same structure as the OpenAI and Azure guides.

---

## 1️⃣ Create Your Anthropic Account & API Key

1. Go to the [Anthropic Console](https://console.anthropic.com/)
2. Sign up with email (Google SSO also supported; no Microsoft/Apple sign-in available)
3. Navigate to **Settings →  [API Keys](https://platform.claude.com/settings/keys)** 
4. Click **"Create Key"**, give it a descriptive name, and copy it immediately

> ⚠️ **The key is shown only once.** If you close the dialog without copying it, you must revoke and regenerate.

Save the key in your `.env` file:

```env
ANTHROPIC_API_KEY="sk-ant-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
ANTHROPIC_MODEL="claude-sonnet-4-6"
```

> 🔑 Claude API keys always start with `sk-ant-`. If yours doesn't match that pattern, it's malformed or from another provider.

---

## 2️⃣ Add Billing & Activate the API

API requests will fail with a `401` or `403` until a payment method is on file.

1. In the [Anthropic Console](https://console.anthropic.com/), go to **Settings → Billing**
2. Add a credit or debit card
3. Purchase a minimum credit top-up to activate your tier (see Tier table below)

### Usage Tiers & Spend Limits

| Tier | Credit Purchase Required | Max Single Purchase | Monthly Spend Limit |
|------|--------------------------|---------------------|---------------------|
| Tier 1 | $5 | $100 | $100 |
| Tier 2 | $40 cumulative | $500 | $500 |
| Tier 3 | $200 cumulative | $1,000 | $1,000 |
| Tier 4 | $400 cumulative | $200,000 | $200,000 |
| Monthly Invoicing | N/A (enterprise) | N/A | No limit |

Tiers advance **automatically** once cumulative purchases reach the threshold. Higher tiers unlock higher rate limits and spend ceilings.

## 4️⃣ Test Your Credentials

Create a simple script (`test_claude.py`) to confirm the connection works:

```python
import os
from anthropic import Anthropic
from dotenv import load_dotenv

load_dotenv()

client = Anthropic(api_key=os.getenv("ANTHROPIC_API_KEY"))

response = client.messages.create(
    model=os.getenv("ANTHROPIC_MODEL", "claude-sonnet-4-6"),
    max_tokens=200,
    messages=[
        {"role": "user", "content": "Write a short story about the Roman Empire."}
    ]
)

print(response.content[0].text)
```

👉 If you see a short story printed, your setup is correct!

### A. Raw API Test Script (with error handling)

To verify the connection robustly, use this extended test:

```python
import os
from anthropic import Anthropic, APIStatusError, APIConnectionError
from dotenv import load_dotenv

load_dotenv()

client = Anthropic(api_key=os.getenv("ANTHROPIC_API_KEY"))

try:
    response = client.messages.create(
        model=os.getenv("ANTHROPIC_MODEL", "claude-sonnet-4-6"),
        max_tokens=200,
        temperature=0.7,
        system="You are a helpful assistant.",
        messages=[
            {"role": "user", "content": "Write a short story about the Roman Empire."}
        ]
    )
    print(f"✅ Connection Successful:\n{response.content[0].text}")
    print(f"\n📊 Token Usage → Input: {response.usage.input_tokens} | Output: {response.usage.output_tokens}")

except APIStatusError as e:
    print(f"❌ API Error {e.status_code}: {e.message}")
except APIConnectionError as e:
    print(f"❌ Connection Error: {e}")
```

> **Note:** The response object exposes `response.usage.input_tokens` and `response.usage.output_tokens` — useful for cost tracking during testing.

## 5️⃣ Available Models & Pricing (May 2026)

All prices are per **million tokens (MTok)**, billed separately for input and output.

| Model | String | Input ($/MTok) | Output ($/MTok) | Best For |
|-------|--------|----------------|-----------------|----------|
| Claude Opus 4.7 | `claude-opus-4-7` | $5.00 | $25.00 | Complex reasoning, agentic workflows |
| Claude Sonnet 4.6 | `claude-sonnet-4-6` | $3.00 | $15.00 | Balanced — recommended default |
| Claude Haiku 4.5 | `claude-haiku-4-5` | $1.00 | $5.00 | High-volume, fast, cheap tasks |

**Cost-saving options:**
- **Batch API:** 50% discount across all models for non-time-sensitive workloads
- **Prompt Caching:** Up to 90% reduction on cached input tokens for repeated context

## 🔍 Tips & Best Practices

- Keep your `.env` file private — **never commit `sk-ant-` keys to version control**
- Track usage and costs in real time at the [Anthropic Console Usage Page](https://console.anthropic.com/settings/usage)
- Monitor your rate limits at [Console → Settings → Limits](https://console.anthropic.com/settings/limits)
- If you hit a `429 Rate Limit` error, your tier's RPM/TPM ceiling has been reached — add a `time.sleep()` or implement exponential backoff
- Use `system` messages to define the model's persona (e.g., `"You are a professional security auditor"`) for consistent behavior
- For async workflows (e.g., Flask), use the `AsyncAnthropic` client with `await` instead of the sync client
