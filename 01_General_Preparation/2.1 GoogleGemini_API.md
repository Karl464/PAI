# 🌐 Configuration & Connection Guide: Google Gemini API

This guide covers account creation, API key generation, credential validation, and basic integration for the **Google Gemini API** — following the same structure as the OpenAI and Azure guides.

---

## 1️⃣ Create Your Google Account & API Key

1. Go to [Google AI Studio](https://aistudio.google.com/)
2. Sign in with your Google account
3. Click **"Get API key"** in the left sidebar
4. Click **"Create API key"** and select or create a Google Cloud project
5. Copy the key immediately

> ⚠️ **Key restriction required from June 19, 2026:** Unrestricted API keys will be blocked by Google. Add API restrictions to your key in the [Google Cloud Console](https://console.cloud.google.com/). Keys inactive for a long period are already being blocked as of May 7, 2026.

Save the key in your `.env` file:

```env
GEMINI_API_KEY="AIzaSyxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
GEMINI_MODEL="gemini-2.5-flash"
```

> 🔑 Gemini API keys start with `AIzaSy`. The SDK accepts either `GEMINI_API_KEY` or `GOOGLE_API_KEY`.

---

## 2️⃣ Add Billing (for Production)

Google AI Studio includes a **free tier** (60 RPM) with no credit card required — enough for prototyping.

For production workloads:

1. Go to the [Google Cloud Console](https://console.cloud.google.com/)
2. Navigate to **Billing** and link a billing account to your project
3. The Gemini API bills **separately** from other Google Cloud services

> 📊 Track usage at [Google Cloud Billing](https://console.cloud.google.com/billing).

## 4️⃣ Test Your Credentials

Create a simple script (`test_gemini.py`) to confirm the connection works:

```python
import os
from google import genai
from dotenv import load_dotenv

load_dotenv()

client = genai.Client(api_key=os.getenv("GEMINI_API_KEY"))

response = client.models.generate_content(
    model=os.getenv("GEMINI_MODEL", "gemini-2.5-flash"),
    contents="Write a short story about the Roman Empire."
)

print(response.text)
```

If you see a short story printed, your setup is correct!

### A. Raw API Test Script (with error handling)

```python
import os
from google import genai
from google.genai import types
from dotenv import load_dotenv

load_dotenv()

client = genai.Client(api_key=os.getenv("GEMINI_API_KEY"))

try:
    response = client.models.generate_content(
        model=os.getenv("GEMINI_MODEL", "gemini-2.5-flash"),
        contents="Write a short story about the Roman Empire.",
        config=types.GenerateContentConfig(
            system_instruction="You are a helpful assistant.",
            max_output_tokens=200,
            temperature=0.7,
        )
    )
    print(f"✅ Connection Successful:\n{response.text}")
    print(f"\n📊 Token Usage → Input: {response.usage_metadata.prompt_token_count} | Output: {response.usage_metadata.candidates_token_count}")

except Exception as e:
    print(f"❌ Error: {e}")
```

> **Note:** Token usage lives in `response.usage_metadata.prompt_token_count` and `response.usage_metadata.candidates_token_count`.

### B. Vertex AI Alternative (Google Cloud-native)

If you prefer Google Cloud ADC / service account auth instead of a raw API key:

```python
import os
from google import genai

client = genai.Client(
    vertexai=True,
    project=os.getenv("GOOGLE_CLOUD_PROJECT"),
    location="us-central1"
)

response = client.models.generate_content(
    model="gemini-2.5-flash",
    contents="Write a short story about the Roman Empire."
)
print(response.text)
```

```env
GOOGLE_CLOUD_PROJECT="your-project-id"
GOOGLE_CLOUD_LOCATION="us-central1"
GOOGLE_GENAI_USE_VERTEXAI=true
```

## 5️⃣ Available Models & Pricing (May 2026)

All prices are per **million tokens (MTok)**, billed separately for input and output.

| Model | String | Input ($/MTok) | Output ($/MTok) | Best For |
|-------|--------|----------------|-----------------|----------|
| Gemini 2.5 Pro | `gemini-2.5-pro` | $2.00 | $12.00 | Complex reasoning, multimodal |
| Gemini 2.5 Flash | `gemini-2.5-flash` | $0.50 | $3.00 | Balanced default — best value |
| Gemini 3 Flash (preview) | `gemini-3-flash-preview` | ~$0.50 | ~$3.00 | Next-gen, currently in preview |

> 🆓 **Free tier:** Gemini 2.5 Flash available at no cost with rate limits (60 RPM) via AI Studio. No credit card needed.

**Cost-saving options:**
- **Batch processing:** Async jobs at reduced rates
- **Context caching:** Significant discounts for repeated large inputs (long system prompts, documents)

## 🔍 Tips & Best Practices

- Keep your `.env` private — **never commit `AIzaSy` keys to version control**
- **Add key restrictions** in Google Cloud Console before June 19, 2026 (unrestricted keys will be blocked)
- Track usage at [Google AI Studio](https://aistudio.google.com/) or [Cloud Console APIs](https://console.cloud.google.com/apis)
- Use `system_instruction` inside `GenerateContentConfig` to define the model's persona
- For async/streaming responses, use `client.aio.models.generate_content(...)` (async client pattern)
- On `429 Resource Exhausted`, implement exponential backoff — RPM quota has been reached
