# ⚡ Configuration & Connection Guide: xAI Grok API

This guide covers account creation, API key generation, credential validation, and basic integration for the **xAI Grok API** — following the same structure as the OpenAI and Azure guides.

---

## 1️⃣ Create Your xAI Account & API Key

1. Go to [console.x.ai](https://console.x.ai/)
2. Sign up with email, Google, or your X account (no X Premium subscription required)
3. Navigate to **API Keys** in the console
4. Click **"Create Key"**, give it a name, and copy it immediately

> ⚠️ The key is shown only once. If you close without copying, you must revoke and regenerate.

Save the key in your `.env` file:

```env
XAI_API_KEY="xai-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"
XAI_MODEL="grok-4-1-fast"
XAI_BASE_URL="https://api.x.ai/v1"
```

> 🔑 Grok API keys always start with `xai-`. The OpenAI SDK reads `XAI_API_KEY` automatically when passed via `api_key`.

---

## 2️⃣ Add Billing & Free Credits

New accounts receive **$25 in promotional credits** upon signup (expire 30 days after account creation).

An additional **$150/month** in credits is available via the data-sharing program (opt in under Billing settings).

For continuous service after credits expire:

1. In the console, go to **Billing → Payment Methods**
2. Add a credit or debit card
3. Set a **hard monthly spending cap** (strongly recommended — tool calls can generate unpredictable costs)
4. Configure usage alerts at 50%, 75%, and 90% of your cap

> 📊 Track consumption in the [xAI Console Usage Explorer](https://console.x.ai/).

## 4️⃣ Test Your Credentials

Create a simple script (`test_grok.py`) to confirm the connection works:

```python
import os
from openai import OpenAI
from dotenv import load_dotenv

load_dotenv()

client = OpenAI(
    api_key=os.getenv("XAI_API_KEY"),
    base_url=os.getenv("XAI_BASE_URL", "https://api.x.ai/v1")
)

response = client.chat.completions.create(
    model=os.getenv("XAI_MODEL", "grok-4-1-fast"),
    messages=[{"role": "user", "content": "Write a short story about the Roman Empire."}],
    max_tokens=200
)

print(response.choices[0].message.content)
```

If you see a short story printed, your setup is correct!

### A. Raw API Test Script (with error handling)

```python
import os
from openai import OpenAI, APIStatusError, APIConnectionError
from dotenv import load_dotenv

load_dotenv()

client = OpenAI(
    api_key=os.getenv("XAI_API_KEY"),
    base_url=os.getenv("XAI_BASE_URL", "https://api.x.ai/v1")
)

try:
    response = client.chat.completions.create(
        model=os.getenv("XAI_MODEL", "grok-4-1-fast"),
        messages=[
            {"role": "system", "content": "You are a helpful assistant."},
            {"role": "user", "content": "Write a short story about the Roman Empire."}
        ],
        max_tokens=200,
        temperature=0.7,
    )
    print(f"✅ Connection Successful:\n{response.choices[0].message.content}")
    print(f"\n📊 Token Usage → Input: {response.usage.prompt_tokens} | Output: {response.usage.completion_tokens}")

except APIStatusError as e:
    print(f"❌ API Error {e.status_code}: {e.message}")
except APIConnectionError as e:
    print(f"❌ Connection Error: {e}")
```

### B. Enabling Live Web Search

Grok's built-in web search and X (Twitter) search are enabled via the `tools` array. These are billed at an additional **$5.00 per 1,000 successful tool calls** on top of standard token costs.

```python
response = client.chat.completions.create(
    model="grok-4-1-fast",
    messages=[{"role": "user", "content": "What happened in AI news today?"}],
    tools=[
        {"type": "web_search"},   # Live web search
        # {"type": "x_search"},   # X (Twitter) search
    ]
)
print(response.choices[0].message.content)
```

> ⚠️ The per-call fee applies only to xAI's built-in tools. Standard function calling with your own tools is billed at normal token rates only.

## 5️⃣ Available Models & Pricing (May 2026)

All prices are per **million tokens (MTok)**, billed separately for input and output.

| Model | String | Input ($/MTok) | Output ($/MTok) | Context | Best For |
|-------|--------|----------------|-----------------|---------|----------|
| Grok 4 | `grok-4` | $3.00 | $15.00 | 256K | Frontier reasoning, complex tasks |
| Grok 4.1 Fast | `grok-4-1-fast` | $0.20 | $0.50 | 2M | Default — cost-efficient, huge context |
| Grok Code Fast 1 | `grok-code-fast-1` | $0.20 | $1.00 | 256K | Code generation, agentic coding |

**Prompt caching is automatic** — no configuration needed. Cache hit prices: Grok 4.1 Fast drops to $0.05/MTok (75% off), Grok 4 drops to $0.75/MTok. Front-load static content (system prompts, documents) to maximize cache hits.

**Batch API:** 50% off all token types for async workloads processed within 24 hours.

**Built-in tool pricing (additional, per 1,000 calls):**
- Web Search: $5.00
- X Search: $5.00
- Code Execution: $2.50

## 🔍 Tips & Best Practices

- Keep your `.env` private — **never commit `xai-` keys to version control**
- **Set a hard monthly cap** in the console — tool calls can generate unpredictable costs without it
- Track usage at [console.x.ai](https://console.x.ai/) under Usage Explorer
- Grok 4.1 Fast's 2M token context means you can load entire codebases or large document sets in a single request
- Switching from the OpenAI SDK is a two-line change (`base_url` + `api_key`) — no other refactoring needed
- On `429 Too Many Requests`, new accounts start with conservative rate limits — implement exponential backoff
